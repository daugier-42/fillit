# fillit

> Résolution de Tetriminos sur une grille — Projet 42

## Description

Le programme lit des formes de Tetriminos (tetriminos) sur l'entrée standard et les place sur une grille carrée minimale sans chevauchement.

Algorithme de placement par backtracking avec représentation bitmap des pièces.

## Utilisation

```bash
make
./fillit < fichier_tetriminos.txt
```

## Compétences

`C` · `Algorithmes` · `Backtracking` · `Bit manipulation`
